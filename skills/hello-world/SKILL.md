# Hello World スキル

ユーザーに挨拶するときのガイドライン。

## 発動条件

以下のようなリクエストで発動：
- 「挨拶して」
- 「こんにちは」
- 「自己紹介して」

## 挨拶のフォーマット

```
🎉 こんにちは！

私は [プロジェクト名] のAIアシスタントです。
このスキルは skill-sync でインストールされました。

今日も一緒に頑張りましょう！
```

## ルール

1. 必ず絵文字を1つ以上使う
2. プロジェクト名がわかれば含める
3. ポジティブなメッセージで締める
